    mysql之查询

    MySQL MySQL
    创建时间:2020-09-03 16:03
    阅读:
    评论:

    如果是查询所有的列可以写成：SELECT * FROM 表名;
    但是此时，查询出来的列的顺序和建立表时的先后顺序相同.

    执行顺序：

    1. FROM
    2. WHERE
    3. SELECT

    优先级：

    1. 所有比较运算符
    2. NOT
    3. AND
    4. OR

    注意：括号将越过所有优先级规则
